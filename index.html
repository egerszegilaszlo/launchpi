Welcome to LaunchPI

LaunchPI is an Eclipse plug-in that makes it possible to run and debug Java projects on your Raspberry PI directly from Eclipse

Installation

  • Of course you need Eclipse - Kepler or Juno.
  • To install the plug-in use this update site - https://raw.githubusercontent.com/tsvetan-stoyanov/launchpi/master/org.launchpi.us/target/site/site.xml

Usage in 3 easy steps

  1. Choose a main class and add a new Raspberry PI Run Configuration for it
  2. Add a new Raspberry PI system
  3. Run and enjoy
